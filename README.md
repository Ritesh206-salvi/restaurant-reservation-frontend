# Restaurant Reservation System - Frontend

React-based frontend for the Restaurant Reservation Management System, providing role-aware interfaces for customers and administrators.

## Live App

https://restaurant-reservation-frontend-six.vercel.app

## Backend Repository

https://github.com/Ritesh206-salvi/restaurant-reservation-backend

## Tech Stack

- React (Vite)
- Tailwind CSS
- React Router DOM
- Axios

## Features

- **Authentication:** Login and Registration pages with role selection (customer/admin)
- **Customer Dashboard:** Browse available tables, create reservations, view and cancel own reservations
- **Admin Panel:** View all reservations, filter by date, cancel any reservation
- **Protected Routing:** Role-based navigation after login (customer → dashboard, admin → admin panel)
